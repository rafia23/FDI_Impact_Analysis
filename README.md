# 📊 Tracing the Impact of FDI on Bangladesh’s Economic Journey (1995–2020)

## 📌 Project Overview
This project analyzes the impact of Foreign Direct Investment (FDI) on Bangladesh’s economic growth, employment, and infrastructure development from 1995 to 2020. Using Power BI, interactive visualizations were created to uncover relationships between FDI inflows, GDP growth, unemployment, and gender-based employment patterns.

---

## 🎯 Objectives
- Analyze trends in FDI inflows over time
- Examine the relationship between FDI and GDP growth
- Explore the impact of FDI on employment and unemployment
- Investigate gender disparities in employment
- Understand how FDI contributes to long-term investments (GFCF)
